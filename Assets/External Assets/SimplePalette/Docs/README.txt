SimplePalette v1.0.0
by: @probe_31

HOW TO USE:
- Download a palette with --PAINT.NET .TXT-- format. (You can get palettes on: https://lospec.com/palette-list)
- Drag the palette file to the SOURCE FIELD and then click on the button LOAD.
- If you want to load other palette, just make right click on the title bar and then click on "Clear Palette"
